## 📥 Installation

Clone the repository:

```
git clone https://github.com/amitroy1992/portfolio-website.git
```

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev
```

## 🌐 Deployment

The project is deployed on **Vercel**.

To deploy:

1. Push code to GitHub
2. Import repository in Vercel
3. Deploy with Vite build settings
